# pillReminder
Minimal app for birth control reminder written in flutter


### Pages
1. Home Page. Current pill package state.
     * Protection state icon.
     * Update pill state.
2. Settings Page. Current settings. 
    * Pill type settings.
    * Alarm settings.
    * App color settings.
3. Calendar Page. 
    * Icons per day.
    * Notes per day.
